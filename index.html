Constructor

new Vue({
    el: '',
    template: '',
    data: {
        props: 'this you can find in templates',
        first_name: "Evan",
        last_name: "You"
    },
    computed: {
        full_name: function(){
            return this.first_name + this.last_name; //Evan You  
        }
    },  
    methods: {
        // life cycle methods
        beforeCreate: function(){}, 
        // checking data, initializing events
        created: function(){},      
        // if (!el || !template)
        beforeMount: function(){},  
        // creating vm.$el and replace el by vm.$el
        mounted: function(){},      
        // item rendered
        beforeUpdate: function(){}, 
        // renders the VDOM again and compares with the real DOM, applies the changes
        updated: function(){},      
        // changes rendered
        beforeDestroy: function(){},
        // complete dismantling of watchers, internal components and event listeners
        destroyed: function(){},    
        // called when the operation stops

        customMethodsAlso: function(){
            //access to data
        }
    }
})

Directives

  • V-bind - dynamically associated with one or more attributes.
  • V-if - condition for rendering the element
    V-else - means “else block” for v-if
  • V-for - loop through an array of objects
  • V-model - connects the state with the input element
  • V-show - toggles the visibility of the element, changing the CSS display property
